# SkyRoute Travel Platform

A flight search and booking platform built with ASP.NET Core (.NET 8) and Angular.

## Project Structure
```
SkyRouteProject/
├── Backend/          # ASP.NET Core Web API
│   ├── SkyRoute.Api/           # Controllers, Middleware
│   ├── SkyRoute.Application/   # Services, DTOs, Validators
│   ├── SkyRoute.Domain/        # Models, Enums
│   ├── SkyRoute.Infrastructure/ # Data, Providers, Auth
│   └── SkyRoute.Tests/         # Unit & Integration tests
├── Frontend/         # Angular application
└── docs/             # Architecture & requirements
```
